# To_do_list with Django
This web app has been developed using the popular Django framework and HTML/CSS for the frontend. My motivation to build this project is so that I can learn about Django and tighten up my skills.This little program has CRUD functionality.
### Basic Features
    
* Register – Users can register and create a new profile
* Login - Registered users can login using username and password
* Social Apps Login – Users can login using their GitHub or Google account
* Read tasks – Users can easily read their tasks 
* create tasks - Once logged in, users can create tasks
* Update tasks – Users can update their tasks 
* Delte tasks – Users can update their tasks 




### Quick Start
To get this project up and running locally on your computer follow the following steps.
1. Set up a python virtual environment
2. Run the following commands
```
$ pip install -r requirements.txt
$ python manage.py migrate
$ python manage.py createsuperuser
$ python manage.py runserver
```
   
3. Open a browser and go to http://localhost:8000/# To-do-list-app-with-user-registration-and-login
